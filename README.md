# Outline

A mod for Rain World that introduces friendly fire mechanics.

## Features

Configure which creatures are considered Ripple Friends and which interactions are modified through the remix menu.

## Installation

1. Download the zip file from [Releases](https://github.com/pjhgamma/ripple-friends/releases).
2. Extract its content in the folder `Steam\SteamApps\common\Rain World\RainWorld_Data\StreamingAssets\mods`.
3. Enable the mod in the remix menu.
